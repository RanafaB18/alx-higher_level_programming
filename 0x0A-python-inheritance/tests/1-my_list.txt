The ``my_list`` module
=====================

Using the ``lookup`` function
-----------------------------

Importing function:
    >>> MyList = __import__("1-my_list").MyList

Appending values:
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]
>>>

Passing a list:
    >>> a = [1, 4, 2, 3, 5]
    >>> my_list = MyList(a)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
